# Iceland Travel Diary — Website Prototyping Project

## Project Overview
Build a pure HTML/CSS project with:
- 25 landing page iterations for the same concept
- 1 gallery page in `index.html`
- No frameworks, no APIs, no backend
- No JavaScript unless absolutely necessary
- Keep everything easy to deploy on Vercel

## Content
- Theme: Iceland travel diary
- Images: provided in project directory (DJI drone photos)
- Verbal descriptions: Claude will write these based on the images

## Design Process
- **v1–v10**: Radically different designs — explore broadly
- **v11–v19**: Combine the strongest ideas from the first phase
- **v20–v25**: Refine one chosen direction — converge and polish
- Each page should feel intentional and distinct

## File Structure
```
index.html          ← gallery page (links to all 25 versions)
v1/
  index.html
  style.css
v2/
  index.html
  style.css
...
v25/
  index.html
  style.css
```

## Code Style
- Separate HTML and CSS files per version
- Use semantic HTML
- Keep file names consistent
- Keep code simple and readable
- No inline styles unless trivial

## Gallery Page (`index.html`)
- Show all 25 versions with previews/links
- Briefly describe what changed in each version
- Clearly mark v25 as the final choice
